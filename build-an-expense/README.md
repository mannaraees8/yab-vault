## Description
Build an expense tracker with AI insights:

FIELDS: description (text, required), amount (number, required), category (select: food/transport/shopping/bills/other), date (date, required), notes (text optional).

VIEWS: Dashboard with monthly totals and AI insights, list view of expenses, create/edit form.

FEATURES: Add, edit, delete expenses. Filter by date/category. AI generates monthly spending summary and simple recommendations.

AI FEATURES: Monthly spending summary in plain English, basic spending pattern insights.

## Data Storage

**Default:** Your data is stored locally in your browser (localStorage)
- Works immediately, no configuration needed
- Data persists between sessions
- Private to your browser

**Optional GitHub Backup:**
1. Go to Settings in the app
2. Enter your GitHub personal access token
3. Enter your GitHub username
4. Data will automatically sync to: `your-username/yab-vault/build-an-expense/data.json`
